# minesweeper-docs

A collection of documentation for Minesweeper that I have gathered.  Mostly copy-pastes, with some occasional small tweaks to wording/phrasing for clarity; some portions are originally written by me.

The goal is to have comprehensive, clearly-written definitions for all minesweeper-specific terminology.

Sources: minesweepergame.com, minesweeper wiki (minesweeper.fandom.com), and various github repos.

Corrections/improvements/additions/questions are always welcome.
